# ChatData 项目记忆

## 产品定位
- **定位**：蔚来汽车内部智能数据分析助手，全场景智能用数工具
- **面向人群**：业务运营人员、管理层、数据分析师等看数人群
- **核心价值**：通过自然语言交互降低数据分析门槛，提升数据查询和分析效率
- **愿景**：让每个人都能便捷进行数据查询、分析和报告生成
- **竞品关系**：与Datasight-BI、Index、飞书小组件形成功能互补

## 核心功能

### 1. 智能问数
- 自然语言查询指标数据
- 支持单指标/多指标查询、时间筛选、维度筛选
- 指标衍生：排序、同环比、占比、排名、二次聚合
- 图表自动生成（柱状图、折线图、指标卡、表格）
- 分析链路可视化（意图识别→解析要素→伪SQL→执行SQL）
- 多轮对话支持

### 2. 智能数据分析
- 维度归因分析（贡献度计算、智能总结）
- 多元线性回归分析
- 异常诊断分析
- 趋势模式分析
- 数据解读与业务洞察

### 3. 报告生成（规划中）
- 多Agent协作：Planner、Data Agent、Analysis Agent、Report Agent、Memory Agent、Knowledge Agent

### 4. 专题配置
- 用户可配置查询和分析提示词、数据源、分析方法、输出格式

## 技术架构

### 核心创新：Text2DSL
- 将自然语言直接翻译为精准的指标DSL，端到端准确率90%+
- 分而治之的思想（任务拆分）+ 多路并行生成 + 智能评估

### 分层架构
1. **语义建模层**：基于维度建模理论，通过本体论构建指标语义
2. **协议与转译层**：指标DSL连接抽象语义与物理数据
3. **核心创新层**：Text2DSL技术
4. **分析闭环层**：从指标DSL到DataFrame，完成归因分析等高阶分析

## 发展规划
- **1期(MVP)**：基础问数能力、飞书机器人、移动端支持
- **2期**：维度归因分析、数据解释归纳
- **3.0期**：用户数据报告、分析范式管理、多Agent协作

## 准确性提升与语义增强（2026-05-13 新增）

### 架构：混合模式
- 前置处理 → 意图预判 → 主模型 → 后置推荐

### 核心模块
1. **输入预处理**：错别字修正、口语规范化、术语对齐（复用问数知识）
2. **意图预判**：小模型分词预判、歧义检测、ABC选项澄清模板
3. **主动澄清**：维度缺失立即澄清、多义词执行后澄清、指标/功能边界反问
4. **推荐生成**：同时间/同维度成员/关联指标推荐，展示在点赞点踩下方

### 澄清策略
- 主动防御型：有歧义立即澄清
- 兜底：指标固有时间周期生成推荐（针对新指标无历史记录）

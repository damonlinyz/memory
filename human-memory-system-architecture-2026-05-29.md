# 人类记忆系统架构设计讨论

## 讨论日期
2026-05-29

## 核心主题
什么样的记忆系统能真正模拟人类记忆的特性：
- 关联网络
- 上下文相关
- 模糊检索
- 情感权重
- 动态演化
- 多模态

## 关键洞察

### 1. 记忆 = 主动构建 + 被动积累

- **主动知识**：用户主要活动/学习中主动获取和记录的
- **被动卷入**：环境碎片（走路时看到的行人、衣服颜色、红绿灯等）

例：过马路等红绿灯时，被动看到的信息（行人颜色、帽子）会在某天被某个场景触发召唤出来。

### 2. 关联是构建出来的

- 构建过程是主动的
- 环境记忆会被连带卷入
- 平时是浅记忆，触发时临时主动构建出来

### 3. 知识在物理世界是可视化的

- 人类记忆形成突触组织
- 记忆系统可能需要某种空间/拓扑性质
- 关联不是查表，而是"距离或方向"的关系

### 4. 触发机制 - 行动流触发

触发不是"搜索"，而是**人在做某件事的过程中**自然发生的。

触发条件 = **关键词 + 场景 + 活动** 三者共同构成

- 当前活动：设计师在做服装设计
- 当前场景：工作坊/对话
- 关键词：红色、衣服

→ 触发历史记忆碎片：某天过马路时看到的红色衣服

关键洞察：记忆不是被"查询"唤醒的，而是被**当前行动流**自动卷进来的。

因此：
- 记录时需要绑定**活动上下文**（不只是内容）
- 检索时基于**当前活动**而非关键词匹配
